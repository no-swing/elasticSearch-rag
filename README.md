# elasticSearch-rag
a Retrieval-Augmented Generation (RAG) model with text retrieval and vector retrieval based on ElasticSearch, with the frontend generated by Gradio.

一个基于Elastic Search的文本召回和向量召回实现的轻量级RAG ，前端由Gradio生成
